# Frontend (React + Vite)

This directory contains the frontend application.

## Quickstart

1. Install dependencies:

   ```bash
   pnpm install
   ```

2. Run dev server:
   ```bash
   pnpm dev
   ```

The dev server runs at:
http://localhost:5173

## Notes

- Use Node.js 20 LTS (see ../.nvmrc)
- If pnpm blocks build scripts, run:
  ```bash
  bashpnpm approve-builds
  pnpm rebuild
  ```
