# 020 — Insights & Budgets ✅ Complete

## Goal

Turn structured transaction data into **useful insights** and introduce spending limits.

## Delivered

### Dashboard
- Monthly KPI cards: income, expenses, net, transaction count
- Top payees bar chart
- Category breakdown chart
- Top 10 individual spendings

### Fixed vs Variable Costs
- Categories tagged as fixed / variable / unclassified
- Breakdown report per month

### Budgets
- Define monthly budget amounts per category
- Budget vs actual comparison with remaining balance
- Overspending highlighted

### Recurring Transactions
- Heuristic detection of recurring transactions (same payee, similar amount, regular interval)
- Mark and review recurring patterns
- Monthly recurring cost KPI on dashboard
